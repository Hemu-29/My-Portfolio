A premium, interactive personal portfolio website showcasing my experience, projects, and skills as a **Software Developer** focused on AI, Machine Learning, and Full Stack Architecture. 

Built with a modern **Spatial Glassmorphism** design language and enhanced with interactive 3D physics using **Three.js**.

👉 **[Live Demo: https://hemu-29.github.io/My-Portfolio/](https://hemu-29.github.io/My-Portfolio/)**

---

## ✨ Key Features

- **Spatial Glassmorphism UI:** Ultra-modern frosted glass interface (`backdrop-filter`) layered over a deep spatial background with ambient, slow-moving neon glow gradients.
- **Interactive 3D Physics Grid:** An advanced background of scattered data blocks (Octahedrons, Torus Rings, Pyramids, Cubes) powered by Three.js. The blocks intelligently **repel** and scatter dynamically from the user's cursor using spring physics and raycasting.
- **Dynamic Content Styling:** Fluid typography pairing (`Outfit` and `Space Grotesk`) with smooth CSS transitions, hover states, and staggered fade-in animations.
- **Fully Responsive:** Flawlessly scales across ultrawide monitors, laptops, tablets, and mobile devices using CSS Grid, Flexbox, and mobile-first media queries.
- **Performance Optimized:** No heavy framework dependencies. Built natively for maximum speed, leveraging hardware-accelerated CSS and optimized WebGL rendering.

---

## 🛠️ Tech Stack

**Frontend Frameworks & Libraries:**
- **HTML5:** Semantic architecture.
- **CSS3:** Custom variables, Glassmorphism, CSS Grid/Flexbox, Keyframe Animations.
- **JavaScript (ES6+):** Core logic and DOM manipulation.
- **Three.js (r134):** Custom 3D WebGL geometries, raycaster collision, and orbital physics.
- **Typed.js:** Dynamic, automated typing effect in the hero header.

**Typography & Icons:**
- **Google Fonts:** `Outfit` (Body), `Space Grotesk` (Headings)
- **Inline SVG Native:** Sharp, lightweight social and external link icons.

## 💻 Local Setup & Development

Want to run the portfolio on your local machine? It requires zero build steps!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hemu-29/My-Portfolio.html.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd My-Portfolio
   ```

3. **Launch the project:**
   Simply double-click on `index.html` to open it in your default web browser. 
   
   *(Alternatively, run it through VS Code using the **Live Server** extension for hot-reloading).*
